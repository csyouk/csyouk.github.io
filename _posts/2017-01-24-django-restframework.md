# Django restframework

django restframework을 쓰기 위해서는 다음 요구사항을 만족시켜야 한다.
- Python (2.7, 3.2, 3.3, 3.4, 3.5)
- Django (1.8, 1.9, 1.10)

다음 dependency들을 설치한다.
~~~
pip install django
pip install djangorestframework
pip install pygments
~~~
