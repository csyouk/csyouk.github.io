1. 포스트 작성 당시의 맥락을 서술한다.
2. 포스트는 제 3자가 읽고도 이해할 수 있는 단어를 선택한다.
3. 정제된 문장으로 정리한다.
4. 작성중인 글은, 나중에 다시 이어서 글을 쓰더라도, 맥락을 놓치지 않기 위해서, 글을 쓰려고 했던 목표와, 키워드를 적어 놓고, 게시 상태를 ```published: false``` 로 바꿔놓는다.
5. 참조한 글들이 있으면, 출처를 반드시 밝힌다.
